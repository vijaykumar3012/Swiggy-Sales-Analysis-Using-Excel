# Swiggy-Sales-Analysis-Using-Excel

📊 **Swiggy Sales Dashboard Analysis using Excel**

Excited to share my latest **Excel Dashboard Project** where I analyzed Swiggy sales data and transformed raw data into meaningful business insights.

### 🔍 Key Insights:

✅ Total Sales: **₹53.01M**
✅ Average Rating: **4.34 ⭐**
✅ Average Order Value: **₹268.51**
✅ Total Orders: **197.43K**
✅ Rating Count: **5.59M**

### 📈 Dashboard Highlights:

* Monthly Sales Trend Analysis
* Daily & Weekly Sales Performance
* Sales Distribution by Food Category
* State-wise Sales Visualization (Map Chart)
* Top 5 Cities by Sales
* Quarterly Sales, Ratings & Orders Summary
* Interactive Slicers for Dynamic Filtering

### 🛠 Tools Used:

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Power Query
* Conditional Formatting
* Slicers
* Map Charts
* Dashboard Design Techniques

### 🎯 What I Learned:

* Data Cleaning & Transformation
* KPI Design and Visualization
* Interactive Dashboard Development
* Business Performance Analysis
* Storytelling with Data

This project helped me strengthen my **Data Analytics and Dashboarding skills** while gaining insights into customer ordering patterns and sales performance.
